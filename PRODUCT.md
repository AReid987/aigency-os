# Product

## Register

product

## Users

Technical founder (expert CLI user, admin access) orchestrating 15+ AI coding agents. Domain expert partner (non-technical, business-focused) contributing strategy and approving milestones. Both share a visual workspace.

## Product Purpose

Aigency OS is a visual-first, agent-orchestrated workspace where a technical founder and domain expert collaboratively design, validate, and build ventures using an orchestra of AI agents. The platform combines venture orchestration, agent communication, quality gates, and knowledge capture into a unified canvas-based interface.

Success: a domain expert can shape business strategy without code; a technical founder can spawn and manage 15+ CLI agents in parallel with full visibility and automatic quality gates.

## Brand Personality

Precise, minimal, technical. Operator-grade confidence. No fluff. The interface should feel like a command center, not a consumer app. Three words: autonomous, precise, founder-spirited.

## Anti-references

- SaaS-cream warm backgrounds (the saturated AI default)
- Bento-box card grids with icon + heading + text repeated endlessly
- Marketing-forward dashboards with hero metrics and gradient text
- Consumer-friendly rounded corners and playful colors
- Emoji in UI elements

## Design Principles

1. **Command center, not dashboard.** Information density serves the operator. Every pixel earns its place.
2. **Agent transparency.** Humans see what agents are doing without understanding internals. Status, progress, and decisions are always visible.
3. **Glass over chrome.** Content sits on translucent glass panels over a deep-space void. Minimal decorative UI.
4. **Sharp precision.** Near-zero border radii. Exact spacing. No softness that signals consumer-grade.
5. **Dark by default.** The environment is a space station. Light themes are not supported.

## Accessibility & Inclusion

WCAG 2.1 AA. All text 4.5:1 contrast against backgrounds. Keyboard navigation for all interactive elements. Respect prefers-reduced-motion. Status conveyed by icon + text, not color alone.
